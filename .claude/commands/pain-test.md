对 `$ARGUMENTS` 做痛苦测试（情感熔炉检查），只诊断、给方向，不代写：

逐项检查并报告是否成立：
1. **情感锚点**：第一幕是否展示了主角最热爱/最依赖的事物（舒适区）？
2. **不公打击**：危机是否由不可抗力/偏见造成、把主角推入孤立境地？而不是轻飘飘的世俗输赢。
3. **赌注升级**：失败代价是否够重（精神死亡/孤独/失去最爱），而非小麻烦？
4. **缺陷自燃**：危机是否由主角自身偏执放大，而非天降反派？
5. **熔炉抉择**：高潮是否逼主角舍弃 Want、换取 Need 的成长？
6. **完美主角检查**：主角是否有真实致命缺陷，而不是无痛通关的圣母？

输出：每项 ✓/✗ + 为什么 + 不成立项的改进方向（非处方）。遵守 CLAUDE.md。
